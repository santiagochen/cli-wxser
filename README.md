### Wxser 命令行工具
 
1. 安装 ： ``` npm install -g wxser  ```
    - ![安装示意图](https://github.com/santiagochen/wxser/blob/master/assets/install.gif?cb1)
2. 使用命令 ：
    - ![使用示意图](https://github.com/santiagochen/wxser/blob/master/assets/paste.gif)
    - ``` wxser ls ``` 列出所有元素
    - ``` wxser get <type> ``` 指定查看某类元素的二级类型
